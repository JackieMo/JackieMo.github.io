#### 一、安装 node、watchman、react-native-cli。在Mac上我们可以通过[HomeBrew](https://brew.sh/)进行相关的安装。

```
1. brew install node
2. brew install watchman
3. brew install flow
4. npm install -g react-native-cli
```

#### 二、ios 开发环境搭建

```
1. react-native init AwesomeProject 
2. cd AwesomeProject
```
